# 慶祝 MDN 20 週年

> 原文發佈日期：2025-07-23,作者 Joe Walker。

## 重點摘要

- MDN 本月迎接 20 週年。最初是社群驅動的 wiki,以協助開發者導覽快速演進的網路、強調 web standards 為主。
- 目前規模:近 14,000 頁文件、超過 33,000 篇在地化文章、近 18,000 個功能的瀏覽器相容性 (compatibility) 資料。
- 定位:面向各程度開發者,推動一個開放的網路生態,為數十億使用者服務。

## 業界傳統:互送蛋糕

瀏覽器廠商間有一個長期傳統 —— 在重大里程碑互送蛋糕。文中提到的例子:

- Microsoft 在 Firefox 2、3、4 推出時各送了 Mozilla 一個蛋糕。
- Mozilla 也回送過一個慶祝 IE10。
- 這次 20 週年由 web.dev 團隊送 MDN 一個蛋糕。

作者藉此強調「即使廠商彼此競爭,最終都是在共建一個大家共用的開放網路」。

## 社群數據

- 每月觸及數百萬開發者。
- 超過 100,000 位 contributor 透過 MDN GitHub organization 互動過。

## 行動呼籲

讀者可在 X / Mastodon / Bluesky 標記 MDN 分享回憶,或直接加入 web platform docs 的協作。
